程式的核心功能
這是一個「食物菜單資料管理工具」，主要用來：

讀取CSV檔案（Excel表格）

清理資料（修正錯誤格式）

存入資料庫

從資料庫讀取資料

主要工作流程
第一步：設定環境
python
# 設定 Django 資料庫環境
# 這樣才能使用 menu.models 裡的資料表

第二步：清理資料（最重要的一步）
程式會處理以下常見問題：

清理文字：移除特殊符號如 @#$% 等

修正錯誤：把「午餐晚餐」改為「午餐,晚餐」

處理價格：把「100元」轉成數字 100.0

處理熱量：把「300卡」轉成數字 300

第三步：存入資料庫
把清理好的資料存入三個資料表：

Dish（菜餚）：菜名、價格、熱量

Category（食材類別）：如蔬菜、肉類

MealTime（供應時段）：如早餐、午餐、晚餐
